Arasu

A Full Stack Web Framework for Go & Dart

Arasu make it easy to build custom, enterprise-grade apps . It is designed to mimic the MVC pattern of frameworks like Ruby on Rails, but with support for the requirements of modern apps: data-driven APIs with scalable, service-oriented architecture. It's especially good for building multi scale, realtime web applications.
