Getting Started

Note: arasu development only work on dart enabled browsers like dartium or dart enabled chrome browser.

Installation

To install the latest stable release with the command-line tool:

go get github.com/arasuresearch/arasu

Creating a New Sails Project

Create a new app:

arasu new demo
cd demo
arasu ds rd generate scaffold User name password age:integer
arasu ds bd generate scaffold Book title:string content:blob price:decimal
arasu ds rd migrate // migrate relational database like mysql table
arasu ds bd migrate // migrate bigdata like hbase table

Now start the server:

arasu server

At this point, if you visit (http://localhost:4000/) you will see the default home page. Now, let's get Sails to do cool stuff.
