Arasu

An Lightning Fast Web Framework

Written on Go, Dart and Angular Dart

Arasu® is an open source web framework.
Arasu is designed to create high performance web apps.
Arasu is especially good for building multi scale, API driven web applications.

Why Arasu?

Arasu is designed to create highly performanced web apps, with the support for the requirements of modern apps like data-driven APIs with scalable, service-oriented architecture. Arasu make it easy to build multi scale,API driven ,realtime enterprise grade web apps.

Go

Server side component of Arasu is written on top of go language. arasu apps are extreamly fast because of goroutine (a lightweight thread by the Go runtime Engine).

Dart

Client side component of Arasu is written on top of dart language. client side component uses angulardart by default.

Data Storage

Arasu uses both RDBMS and BIGDATA for server side storage. default RDBMS is mysql. default BIGDATA is Hbase. Arasu uses indexedDB for client side storage.

Arasu on Dart

there is another port of arasu server component arasuondart is re-written in dart language . but, it is at pre alpha stage .even not ready for development.

Testing

Arasu use builtin test package for unit test ( for both go and dart). Arasu is implementing a BDD framework arasu bdd to support BDD on selenium with tools like capybara and spinach.
